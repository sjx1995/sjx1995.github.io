变量

定义变量要用var关键字，定义的同时还可以赋值。
局部变量：定义在函数中的变量，只能在该函数中使用。
全局变量：直接在script元素中定义的变量，可以在任何地方使用。

基本类型

javascript基本类型有三个：

  • 字符串类型(string)
  • 数值类型(number)
  • 布尔类型(boolean)

字符串类型

表示字符串时要用引号。

布尔类型

布尔类型只有true和false两个值。

JavaScript 运算符

算数运算符

运算符 描述
+、-、*、/ 加、减、乘、除
% 求余
++ 累加
– 递减

赋值运算符

运算符 例子 等价于
= x=y
+= x+=y x=x+y
-= x-=y x=x-y
*= x*=y x=x*y
/= x/=y x=x/y
%= x%=y x=x%y

比较运算符

比较和逻辑运算符用于测试true或false。

运算符 描述
== 相等(仅比较值)
=== 相同(比较值和类型)
!= 不相等
!== 不相同
>、>= 大于、大于或等于
<、<= 小于、小于或等于
